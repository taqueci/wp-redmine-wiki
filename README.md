# WordPress Redmine Wiki plugin

This plugin embeds Redmine wiki page.

## Usage

Add the shortcode to a post or a page.

Example:

    [url="https://example.com/redmine" project="foo" wiki="Wiki" key="243f6a8885a308d313198a2e03707344a4093822"]
